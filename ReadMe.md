## Dataset
### 1. Billboard songs dataset
`billboard200.csv` can be downloaded from Kaggle.com (https://www.kaggle.com/datasets/ludmin/billboard) 
> **Note:** This project only use songs from 2025 for analysing song lyrics of the recent year
  
### 2. Musical songs dataset
`musical.csv` has a musical songs lyric manually collected from https://genius.com/ which includes the following musical titles:
-  Be More Chill
-  Beetlejuice
-  Dear Evan Hansen
-  EPIC
-  Frozen The Musical
-  Hamilton
-  Heathers The Musical
-  In The Heights
-  Mean Girls The Musical
-  Ride the Cyclone
-  Tangled
-  Waitress
-  Wicked
-  Wicked (Movie)
  
This dataset has been uploaded to the following link for study purposes:
